# Twister Caller

An automated twister caller for people too lazy to use the spinner.

## Usage

Windows:

```
./twister.ps1
```

Other:

```
Too bad
```

## Help

Some help tips

### Doesn't work

Remove this line and try again

```
$speak.SelectVoice('Microsoft Zira Desktop')
```

### Too fast

Change the value on this line

```
    Start-Sleep -s 3
```
